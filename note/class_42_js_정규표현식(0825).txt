html과 달리 js에서는 패턴 앞뒤에 / (슬래시) 가 붙음.
var patternNum = /[0-9]/;
var patternEng = /[a-zA-z]/;
var patternKor = /[가-힣ㄱ-ㅎㅏ-ㅣ]; (자음, 모음만 있는 경우에도 한글로 취급)
특수문자의 경우는 일부 ( .  \  [] -  ) 그대로 사용하면 적용되지 않은 것들이 있으므로 \를 붙여 구분함.

--Array 내부 객체 함수
배열을 정렬하는 함수로는 sort가 있음. arr.sort();
그러나 arr의 원본을 저장하는 arr_original = arr 을 만들고 arr.sort()하더라도
arr_original이 함께 정렬됨. (값 자체를 가져오는 것이 아니라, arr의 주소를 가져오는 것이기 때문)

또한 sort 기능은 ascii코드를 기준으로 정렬하는데, 문자 정렬은 문제가 없으나 숫자 정렬은 앞자리를 기준으로 비교함.
(&&  1 100 50 9 가 있는 경우, 1 - 100 - 50 - 9 순서로 정렬됨)
따라서 정렬을 위해 새로 메소드 생성. callback함수를 통해 가능함.
arr.sort((left,right) => left-right); // 오름차순 정렬
(두 매개변수를 서로 뺄셈하여 리턴한 값이 음수인 경우는 자리를 바꾸지 않도록 하여 정렬이 가능함.
기타 .reverse (자리 순서를 뒤집음) , .slice (0,3) (배열의 일부를 잘라냄. 0,3인 경우 0부터 3번째 앞(2번째)까지)

--Date 내장 객체 함수
new Date().getTime()등을 사용 가능. (getYear, getMonth등으로 데이터를 따로 추출할 수도 있다)
다만 사용시 형식은 다소 불편할 수 있으므로, 이 역시 유저가 직접 간단한 함수를 짜서 조정할 수 있음.
(ex) 특정 두 기간의 사이를 구하는 함수.. Math.abs(new Date() - new Date(특정일)) 한 후 return에 (날짜) + 일 등으로 설계가 가능.
