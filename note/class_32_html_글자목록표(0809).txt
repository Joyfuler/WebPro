head 영역은 페이지 설정 관련 부분.
body 영역 안에는 웹문서에 실제로 출력되는 정보를 입력.
엔터/ 탭 / 스페이스 여러 개 등은 모두 스페이스 1개로 인식함. 따라서 <br /> 을 사용해 줄을 바꾸고,
&nbsp; 를 이용해 스페이스를 여러 개 넣을 수 있음.
변수선언은 let을 사용. let name = prompt("이름은?", "홍길동");
document.write(name + "님 반갑습니다<br>");

소스가 길어지는 경우 새로운 파일을 만든 뒤 해당 파일을 가져오도록 함.
<link href = "2.css" rel = "stylesheet" type ="text/css" />

<h1 id="top"> h1은 제목태그. ="top"는 가장 위에 위치한 것이며 유일해야 함. (자동개행 + 큰굵은글씨) </h1>

<>를 직접 텍스트로 입력하고 싶다면 (태그혼동방지) &lt; &gt;로 입력 (less than, greater than)
<a href = url링크> 해당링크이동 </a> 
<img src = 이미지주소 alt = "그림이 표시 안될 경우 표시될 텍스트"> 
<b> 진하게 <del> 글씨가로줄 

스타일 태그 관련
<span> style 태그 내 span{} 안에 color등 색을 지정하고, 해당 부위만 다르게 출력이 가능.
혹은 <body> 내에서 <span syle="color:red"> </span>으로 해당 줄만 바꿔 출력이 가능.
background-color : red; (웹화면배경색지정)
color : #000000; (글씨의 색깔지정)

블럭형태그: 해당하는 태그를 붙이면 그 내용은 자동줄바꿈이 됨. (항상 다음줄에 태그생성) <ul><h1>...
인라인태그: 해당하는 태그를 붙이고 다음 내용을 입력할때 자동줄바꿈이 없음. <a><img><script>...

**rowspan을 사용한다면 먼저 rowspan의 영향에 들어가는 가로축(td)을 해결하고, 그 후 세로축(tr)을 설정할 것.
